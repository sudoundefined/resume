# Deepanshu Srivastava - Professional Resume

This repository contains my ATS-optimized professional resume formats, tailored for Technical Lead and Senior Software Engineering roles.

## File Formats Available
* **Deepanshu_Srivastava_Resume.pdf / .docx / .html**: The complete, comprehensive version of my resume detailing all projects, metrics, and technical expertise over the last 9+ years.
* **Deepanshu_Srivastava_Resume_2Page.pdf / .html**: A concise, punchy version specifically formatted and styled to fit perfectly onto two pages when printed or converted to PDF, without losing any key metrics or ATS keywords.
* **Deepanshu_Srivastava_Resume_ATS.pdf / .html**: Versions strictly adhered to Applicant Tracking System (ATS) parsing rules for maximum compatibility with enterprise HR systems (Workday, Greenhouse, etc.).

## Technologies Highlighted
* Node.js, React.js, Angular, TypeScript
* Distributed Systems & Microservices (SOA)
* Multi-Tenancy & Micro-frontends (Single-SPA)
* CI/CD, Docker, Kubernetes

## Usage
The `.pdf` and `.docx` formats are fully prepared and ready for direct submission to Applicant Tracking Systems. If you wish to edit them, use the underlying `.md` or `.html` source files.
