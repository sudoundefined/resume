# Deepanshu Srivastava - Professional Resume

This repository contains my ATS-optimized professional resume formats, tailored for Technical Lead and Senior Software Engineering roles.

## File Formats Available
* **Resume-Final.md / .html**: The complete, comprehensive version of my resume detailing all projects, metrics, and technical expertise over the last 9+ years.
* **Resume-2Page-Optimized.md / .html**: A concise, punchy version specifically formatted and styled to fit perfectly onto two pages when printed or converted to PDF, without losing any key metrics or ATS keywords.
* **Resume-ATS-Optimized.md / .html**: Versions strictly adhered to Applicant Tracking System (ATS) parsing rules for maximum compatibility with enterprise HR systems (Workday, Greenhouse, etc.).

## Technologies Highlighted
* Node.js, React.js, Angular, TypeScript
* Distributed Systems & Microservices (SOA)
* Multi-Tenancy & Micro-frontends (Single-SPA)
* CI/CD, Docker, Kubernetes

## Usage
To generate the final PDF for applications, simply open any of the `.html` files in a web browser (e.g., Google Chrome) and use the native Print functionality (`Ctrl+P`) to "Save as PDF".
