# Deepanshu Srivastava - Portfolios Repository

Welcome to the central repository for my personal portfolios and resume websites. This repository contains two distinct versions of my portfolio, showcasing different web development paradigms and design aesthetics, alongside my traditional resume files.

## Repository Structure

- **`/` (Root Directory)**: The main, ultra-premium Vanilla HTML/CSS/JS portfolio. Built with a focus on raw performance, striking "Abyssal Tech" aesthetics, and no heavy frameworks.
- **`/3d-portfolio`**: A React + Three.js + Framer Motion interactive 3D portfolio.

---

## 1. The Premium Vanilla Portfolio (Root)

A highly polished, dark-themed "Abyssal Tech" resume built purely with standard web technologies. 

### ✨ Features
- **Abyssal Tech Theme**: Deep dark backgrounds with glowing neon gradients (`#00f0ff` & `#ff00ff`).
- **Advanced Glassmorphism**: Frosted glass panels with subtle borders and shadow drops.
- **GSAP Animations**: Smooth scroll-triggered reveals and timeline animations.
- **Custom Interactive Elements**: JS-powered 3D tilt cards and a dynamic mouse-tracking glow.
- **Mobile-First Responsiveness**: Handled flawlessly using Tailwind CSS v4 via CDN.
- **Working Contact Form**: Integrates directly with [FormSubmit](https://formsubmit.co) for serverless email transmission.

### 🚀 How to Run
Since this is built with pure Vanilla HTML/CSS/JS, **no build step is required!**
1. Simply clone the repository.
2. Open `index.html` directly in any modern web browser.
3. *Optional*: Use VS Code's "Live Server" extension for a hot-reloading development experience.

---

## 2. The 3D React Portfolio (`/3d-portfolio`)

A modern, highly interactive React application leveraging Three.js for 3D elements and Framer Motion for complex layout animations.

### ✨ Features
- **React 19 & TypeScript**: Built with the latest React version for robust, type-safe development.
- **Three.js Canvas**: Interactive 3D scenes integrated via `@react-three/fiber` and `@react-three/drei`.
- **Framer Motion**: Fluid, spring-based animations and layout transitions.
- **Vite & Tailwind CSS**: Lightning-fast build tooling and utility-first styling.

### 🚀 How to Run
This project requires Node.js to be installed.
```bash
cd 3d-portfolio

# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build
```
